Портативная (переносная) версия дистрибутива Wwise
Автор: night_dragon_on

===============================

Состав пакета:

Wwise v2015.1.9 build 5624
Microsoft Visual C++ 2013 x86

===============================

core_program.dat - ядро (основной контейнер) дистрибутива
Wwise.bat - ярлык для запуска редактора Wwise
WaveViewer.bat - ярлык для запуска просмотрщика для *.wav файлов
FilePackager.App.bat - ярлык для запуска дополнения для перепаковки банка в *.pck контейнер
delete_settings.bat - при запуске удаляет локальные настройки программы на ПК пользователя
